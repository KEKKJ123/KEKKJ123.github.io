Hexo Github 博客接入搜索引擎

在默认情况下，搜索引擎不会主动收录我们的网站。

那如何才能让读者看到我们的文章呢？

阅读本篇教程，相信你一定会找到想要的答案！

零、部署前准备

为了减少部署过程的麻烦，作者建议读者准备科学上网工具。

本文不涉及百度搜素引擎，因为 Github 禁用了百度爬虫，基于 Github Page 的博客网站无法被收录。

Fluid 主题原生支持 Google Analytics 导入。

为什么使用 Bing 和 Google 搜索引擎？

一、Google Analytics

Google Analytics 是 Google 旗下的网络分析服务平台。

1.创建账号获取 ID

① 通过谷歌账号登录 Google Analytics

② 点击左上角创建账号，选择账号（第一次必须要创建账号），填写下方的账号名称（随便填）

③ 填写媒体资源名称，行业类别选择其他业务活动，业务规模选择小型

④ 业务目标选择其他，点击创建

⑤ 数据收集选择网站，填写自己的博客网站网址，并填写数据流名称

⑥ 点击刚才创建的数据流，在这里面可以看到衡量 ID（即 measurement_id）：G-XXXXXXXXXX。

2.开启网页访问统计

进入博客根目录，编辑 _config.fluid.yml，找到网页访问统计（web_analytics）并启用和填入衡量 ID。

二、Google Search Console

Google Search Console 是 Google 搜索引擎的网站管理员工具，你可以在这里调整你的个人网站。

① 添加

三、Bing Webmaster Tools

Bing Webmaster Tools 是 Bing 搜索引擎的网站管理员工具，你可以在这里调整你的个人网站。

1.注册和登录

进入 Bing Webmaster Tools，点击右上角登录，你可以选择 Microsoft、Google 和 Facebook 三种方式登录。

这里选择第一种。你也可以将 Microsoft 账号与 Google 账号绑定，便于后续的同步。

2.导入网站

如果是首次登录，会提示添加你的个人网站。

这里有两种导入方法：Google Search Console 导入（推荐）和手动添加网站。

第一种需要有 GSC 账号并已导入网站，而它和 Bing Webmaster Tools 都是网站管理员工具。

第二种则是直接输入网站根目录。例如我的网站 https://kekkj123.github.io/。

如果想要 Google 也收录你的网站，最好选择第一种方法，会节省不少时间。

作者选择了第一种方法。如果只想导入至 Bing，可以参考这篇文章。
