在默认情况下，搜索引擎不会主动收录我们的网站。

那如何才能让读者看到我们的文章呢？

阅读本篇教程，相信你一定会找到想要的答案！

一、全球

一、Bing Webmaster Tools

Bing Webmaster Tools 是 Bing 搜索引擎的网站管理员工具，你可以在这里调整你的个人网站

1.注册和登录

进入 Bing Webmaster Tools，点击右上角的登录，你可以选择 Microsoft、Google 和 Facebook 三种方式登录。

作者这里选择了第一种。后续你也可以将 Microsoft 账号与 Google 账号绑定，这样就不会意外创建新的账户了。

2.导入网站

如果是首次登录，会提示添加个人网站。这里有两种导入方法：Google Search Console 导入和手动添加网站。

第一种需要你有 GSC 账号，而它和 Bing Webmaster Tools 都是网站管理员工具。

第二种则是直接输入网站根目录。例如我的网站 https://kekkj123.github.io/。

如果你想要 Google 也收录你的网站，最好先注册一个GSC账号，
